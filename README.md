## Features
- Only 3 layers!
- No combos, no one-shot modifiers
- No tap dance
- No home row mods
- Reliable layer mappings for numbers, symbols, and function keys
- Colemak keyboard layout (files for QWERTY layout available)

## Rationale
I have been using this layout for my Corne v3 for over 4 years, and there was pratically **little to no learning curve** adapting to this from a regular row-staggered keyboard, although I did have a couple years of touch typing before.\
AltGr is on the main layer because I type in different languages.

For the **Sym** layer, I included a numpad on the left side and the `-` key on the home row for minimal travel distance. It works really well for TypeRacer texts.\
The parentheses `{ [ ( ) ] }` are put symmetrically on the right side, and the rest of the `Shift` + (Number) symbols are mapped ergonomically too.

For the **Func** layer, the 12 F keys are mapped intuitively on the left. `Caps Lock` is added to both `Shift` keys. Volume control and general navigation keys are on the right side, mapped very intuitively too.\
Since there are no one-shot modifiers, there are other mappings to include some key combos that would otherwise be impossible to execute.
- **Func** + `G` = `GUI` + `Space` (switches keyboard languages)
- **Func** + `B` = `Alt` + `Tab` (switches tabs)
There are also two convenient macros added for navigating through your browser:
- **Func** + `L` = `Ctrl` + `Shift` + `Tab` (cycles through open tabs counterclockwise)
- **Func** + `Y` = `Ctrl` + `Tab` (cycles through open tabs clockwise)

Additionally, the `Print Screen` key is strategically mapped on **Func** + `D` for taking quick screenshots when gaming.
